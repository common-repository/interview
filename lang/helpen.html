1. You create an interview
  2. You invite by email user(s) for the interview.
  3. You publish the interview once users have answered to the interview's questions.

Some extra features:

Request an interview to an user (you ask an user to answer to your interview questions)

Request an interview means an user will have to answer to questions : to integrate an interview into one of your pages or posts, just copy / paste shortcode: [interview id=number]

ex: for interview number 3, paste:

[interview id=3]

An user must be registered and logged on to pass an interview. Guest users cannot pass an interview. That's why when you invite someone, you can create an new WP user for this person.

Publish / playback a recorded interview: once interviewed finished, publish its content into your post or page.

That means that you can publish an interview an user have done on a page or on post.

You can simply create a new post with interview inside simply by clicking the button "publish [email] interview"

You can also do it manually by using the following shortcode

ex: user whose email is user@email.com has completed interview number 3

[interviewPublish id=3 email=user@email.com]
